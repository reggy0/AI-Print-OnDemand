## 🗝️ Key Features 👇

* 🔐 Authentication (Email + Google with Better Auth)
* 🛍️ Product listing system
* 🎨 Advanced design editor (drag & drop elements)
* 🤖 AI artwork generation from prompts
* 🖼️ Merge artwork directly on mockups
* 💳 Stripe checkout with webhooks
* 📦 Order handling system
* 🔄 Background removal (RemoveBg API)
* ☁️ Media upload & storage (Cloudinary)
* 🚀 Deployment-ready (Render)
* 💻 Built with MERN (MongoDB, Express, React 19, Node.js)
